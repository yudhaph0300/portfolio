Booking Office System - Internship and Independent Study (MSIB) Merdeka Campus Alterra Academy
I have participated in a program organized by the government in collaboration with Alterra Akademi which includes internship programs and independent study. In this program, I choose to follow Understand Everything About Vue JS Front-End Engineer Program. This program is designed to provide a thorough understanding of the Vue JS framework, from project creation to project deployment. In addition, at the end of the program, there is a collaboration between this program and other programs held by Alterra Akademi, such as UI/UX, Backend (Golang), Flutter, and Quality Control.
This booking office is a website that can be used by users to rent an office. The user will be presented with all data from the office, which will have a button to view details from the office. In the office details the user can see the location of the office via the map and choose the day and time the user will rent the office, and the system will automatically display the price that must be paid by the user. From renting the office, users can also add reviews that can later be seen by other users.
From the Front-end (Vue JS) side, we use several tools to build this system, including:
  • Vue CLI
  • Vue-router
  • Vuex
  • Vue2-leaflet
  • Vuetify
  • Axios
  • JSON-server
  • Netlify
  • The API of Open Street Maps
  • Endpoints provided by Back-end
To organize the program code that has been made we use Github. Later each member of the Front-end division can collaborate on the results of the code according to the job desk that has been given.
