Booking Office
Visit Github
Booking office merupakan sebuah tugas akhir kelompok yang harus dikerjakan saat saya mengikuti pelatihan di Alterra Academy tahun 2022. Dalam pengerjaannya kami dikolaborasikan dengan divisi lain diantaranya UI/UX, Back-end (Golang), Front-end (Vue JS dan Flutter), serta Quality Engineering. Dalam pengerjaan project ini saya masuk kedalam divisi Front-end dengan menggunakan Vue JS sebagai frameworknya.
Booking office ini merupakan suatu website yang dapat digunakan oleh user untuk melakukan penyewaan sebuah kantor. User akan disuguhkan seluruh data dari office yang nantinya ada tombol untuk melihat detail dari kantor tersebut. Didalam detail kantor user dapat melihat lokasi dari kantor melalui map serta memilih hari dan jam berapa user akan menyewa kantor tersebut, dan secara otomatis sistem akan menampilkan harga yang harus dibayar oleh user. Dari penyewaan kantor tersebut user juga dapat menambahkan review yang nantinya dapat dilihat oleh user lain.
Dari sisi Front-end (Vue JS) kami menggunakan beberapa tool untuk membangun sistem ini, diantaranya :
  • Vue CLI
  • Vue-router
  • Vuex
  • Vue2-leaflet
  • Vuetify
  • Axios
  • Json-server
  • Netlify
  • API dari Open Street Maps
  • Endpoint yang disediakan oleh Back-end
Untuk mengorganisasikan kode-kode program yang telah dibuat kami menggunakan Github. Yang nantinya setiap anggota dari divisi Front-end dapat mengkolaborasikan hasil kodenya sesuai dengan jobdesk yang telah diberikan.
